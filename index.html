Here's a complete working GitHub Actions workflow for fetching and syncing news every 5 minutes: **Important note:** GitHub Actions minimum schedule is **5 minutes**, and in practice it often runs every **5-10 minutes** (not exact). ## How to use this: **1. Add the file to your repo:** ``` .github/workflows/main.yml ``` **2. Enable GitHub Pages** (optional, to view your news site): - Go to **Settings → Pages** - Source: **Deploy from branch → main → / (root)** **3. That's it!** The workflow will: | Step | What happens | |------|-------------| | ⏰ Every 5 min | Workflow triggers automatically | | 📡 Fetch | Pulls top news from BBC, CNN, Reuters | | 💾 Save | Updates `news.json` with latest articles | | 🌐 Generate | Rebuilds `index.html` news page | | 🔄 Push | Commits changes back to your repo | **To add more news sources**, just add RSS feed URLs to the `FEEDS` list in the workflow. Most news sites have a free RSS feed.
